# Pilot Program Management

## Pilot Program Framework

### The Goal
Convert free pilot users into paying customers by proving clear ROI.

### Timeline: 12 Weeks

| Phase | Weeks | Focus | Deliverable |
|-------|-------|-------|-------------|
| **Setup** | 1-2 | Onboarding, user selection | Pilot charter |
| **Launch** | 3-4 | Training, first usage | 70% activation |
| **Measure** | 5-10 | Regular usage, feedback | Weekly reports |
| **Review** | 11-12 | ROI analysis, expansion | Contract proposal |

## Week-by-Week Playbook

### Week 1: Kickoff
- [ ] Sign pilot agreement
- [ ] Select 20 pilot users
- [ ] Install app, create accounts
- [ ] Baseline survey (confidence level)
- [ ] Schedule check-in calls

### Week 2: Training
- [ ] Live training session (30 min)
- [ ] Share quick-start guide
- [ ] Office hours for questions
- [ ] Track first logins

### Week 3: Activation Push
- [ ] Daily tips via email/WeChat
- [ ] Highlight streak feature
- [ ] Address technical issues
- [ ] Target: 70% have practiced

### Week 4: First Check-in
- [ ] Group call with all users
- [ ] Collect initial feedback
- [ ] Share usage stats
- [ ] Adjust based on feedback

### Week 5-10: Steady State
- [ ] Weekly email with tips
- [ ] Bi-weekly manager update
- [ ] Monthly user survey
- [ ] Track progress metrics

### Week 11: Pre-Review
- [ ] Prepare ROI analysis
- [ ] Collect testimonials
- [ ] Draft contract proposal
- [ ] Schedule decision meeting

### Week 12: Close
- [ ] Present results to decision makers
- [ ] Propose company-wide rollout
- [ ] Negotiate contract terms
- [ ] Set rollout timeline

## Success Metrics

### Usage Metrics
- **Activation rate:** 70% of users try app
- **Weekly active:** 40% use app weekly
- **Sessions per user:** 3+ per week
- **Average session:** 8+ minutes

### Outcome Metrics
- **Confidence score:** +20% improvement
- **Manager feedback:** 4+ / 5 rating
- **Net Promoter Score:** 50+
- **Feature requests:** Collected for roadmap

### Business Metrics
- **Support tickets:** < 5% of users
- **Churn during pilot:** < 10%
- **Expansion interest:** 80%+ want company-wide

## Common Pilot Pitfalls

### 1. Wrong Users
**Problem:** Selected users don't have the pain point
**Fix:** Screen for international call frequency

### 2. Poor Onboarding
**Problem:** Users don't understand how to use app
**Fix:** Live training, not just email

### 3. No Executive Buy-in
**Problem:** Decision maker not engaged
**Fix:** Include them in monthly reviews

### 4. No Clear Success Criteria
**Problem:** Can't prove value at end
**Fix:** Define metrics in Week 1

### 5. Poor Communication
**Problem:** Users forget about app
**Fix:** Regular reminders, tips, check-ins

## Converting to Contract

### The Conversation (Week 12)

**Opening:**
"Over the past 3 months, 20 of your employees used Culturalbility. Here's what happened..."

**Present Results:**
- Usage metrics (screenshots of dashboard)
- Confidence improvement (before/after scores)
- Testimonials (quotes from users)
- Manager feedback (interview summaries)

**The Ask:**
"Based on these results, we'd like to propose a company-wide rollout. Here's what that looks like..."

**Handle Objections:**
- Price: "ROI is clear - one avoided miscommunication pays for annual license"
- Timing: "We can phase rollout by department"
- More pilots: "We can start with one more team as proof"

### Contract Negotiation

**Standard terms:**
- 12-month commitment
- ¥1,500/employee/year
- Volume discount at 100+ employees
- Quarterly business reviews

**Be flexible on:**
- Payment terms (quarterly vs annual)
- Start date (align with fiscal year)
- Pilot extension (if needed)

**Don't negotiate:**
- Price for small deals (<50 employees)
- SLA/support levels
- Data ownership
